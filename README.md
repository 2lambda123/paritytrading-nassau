Nassau
======

Nassau implements NASDAQ transport protocols on the JVM.

Download
--------

Add a Maven dependency to Nassau:

    <dependency>
      <groupId>org.jvirtanen.nassau</groupId>
      <artifactId>nassau</artifactId>
      <version><!-- latest version --></version>
    </dependency>
