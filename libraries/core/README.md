Nassau Core
===========

Nassau Core implements the NASDAQ transport protocols.


Download
--------

Add a Maven dependency to Nassau Core:

    <dependency>
      <groupId>com.paritytrading.nassau</groupId>
      <artifactId>nassau-core</artifactId>
      <version><!-- latest version --></version>
    </dependency>
